Documentation Links for the SMART on FHIR Javascript Libraray

The tutorials presented here make extensive use of the SMART on FHIR Javascript Libraries. The libraries used here are the ones recommended by the Cener smart-on-fhir-tutorial project (see details below).

Cerner SMART on FHIR Tutorial This is a tutorial provided by Cerner. This is the tutorial that was used as a starting point for these examples. (https://engineering.cerner.com/smart-on-fhir-tutorial)
The github repository for the Cerner tutorial is at https://github.com/cerner/smart-on-fhir-tutorial
This repository was forked. This forked version was used as a template for the tutorials presented here. The forked version used is at https://github.com/NACHC-USA/cerner-smart-on-fhir-tutorial
Cerner Sandbox This is the Cerner sandbox where you can test your SMART on FHIR application. You will need to create and account to use this sandbox.
SmartHealthIt Sandbox This is the sandbox where you can test your SMART on FHIR application. You can use this sandbox out of the box without creating an account.
SMART on FHIR Home page This is the main page for the SMART project (http://docs.smarthealthit.org).
SMART on FHIR JavaScript Library This is the home page for the documentation for the SMART on FHIR JavaScript Library used in the tutorials presented here.
