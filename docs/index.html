HIV-CDS Github Repository

This is the documentation page for the hiv-cds project from cqframework. This implementation guide provides technical artifacts to aid in the implementation of appropriate HIV screening. The artifacts include data element definitions, data collection questionnaires, and logic for determining HIV risk and appropriate screening and follow-up. The Implementation Guide that is built from this project can be found at https://build.fhir.org/ig/cqframework/hiv-cds/recommendation-screening.html The Plan Definition that is built from this project can be found at https://build.fhir.org/ig/cqframework/hiv-cds/PlanDefinition-HIVScreening.html
